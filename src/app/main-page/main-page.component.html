Below are some examples of such new technologies I learned to create this project and what I did with them.


  1. Amazon Web Services (AWS)

    • Registered my website domain via Route 53 and retrieved my SSL certificate from the AWS certificate manager

    • Utilized Cloudfront to allow for a faster content delivery network

  2. Angular 8 / Bootstrap

    • Learned Angular 8 from scratch and read much of the documentation which proved invaluable

    • With that knowledge I created highly modularized components and followed standards for encapsulated component CSS

    • Utilized the Bootstrap library to create a more responsive components

    • Component prototypes were built quickly using this method which lead to smoother development
